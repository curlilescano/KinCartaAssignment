
# Kin + Carta Assignment
This are the instructions in order to run this scripts.

Local computer set up process:
- You need to have chrome (and/or firefox) installed. Chrome has been set as default, but if I want to run it on Firefox just change the "browser" property in config.properties file to "firefox"
- Also you need to have java and maven installed.
- And lastly you need to execute the following command in order to run the test "mvn test"
